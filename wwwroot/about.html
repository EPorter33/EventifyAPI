About Eventify

Eventify is a simple and intuitive platform designed to help users create, manage, and explore events across your health authority in the NHS. It demonstrates a full-stack implementation using .NET 10, SQL Server, and modern front-end design principles.

About the Developer

This project was developed by Elizabeth Porter, focused on full‑stack web development. It showcases skills in systems analysis, database integration, API development, and user‑centred design using the NHS.UK Frontend for accessibility and professional styling.
